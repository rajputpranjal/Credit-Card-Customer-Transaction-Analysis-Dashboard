# Credit-Card-Customer-Transaction-Analysis-Dashboard

# Project Overview

This project focuses on analyzing credit card customer and transaction data to generate meaningful business insights. The dashboard helps understand customer behavior, revenue trends, and transaction patterns for better decision-making.

# Objectives
Analyze total revenue, interest earned, and transaction volume

Understand customer demographics and behavior

Identify spending patterns across different categories

Evaluate transaction trends and card usage types

# Tools & Technologies
Oracle Database

Microsoft Power BI

SQL Queries

Data Cleaning & Transformation

Data Modeling

DAX (Data Analysis Expressions)

# Project Workflow
Imported raw dataset into Oracle SQL Database

Performed data extraction and transformation using SQL queries

Connected Power BI to Oracle Database

Created data model and relationships

Built DAX measures for advanced calculations

Designed an interactive dashboard for insights
